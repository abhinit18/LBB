  • 1. The output of this code will be undefined for the first time.
    The reason for this is, localStorage will be null as it doesn't gets any value.

  • 2. Due to asynchronous nature of http request, it takes time to get response but simultaneously localStorage is already being executed null and hence,

    loadData = localStorage.getItem(“autocomplete”).

    which is null.

But after a while refreshing the page, localStorage sets the value after receiving response from http request
and therefore set the value of loadData = localStorage.
Such that we will not get the result of undefined.
